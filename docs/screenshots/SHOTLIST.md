# Screenshot Shot List

This is the minimum screenshot set for a clean public-facing Scribbles-N-Scripts milestone.

## Core Product

1. Welcome screen with recent projects
2. Main workspace in linear mode
3. Modular mode in corkboard presentation
4. Modular mode in outliner presentation
5. Project Settings with appearance presets visible

## Writing Workflow

1. Find / Replace with grouped results and replace preview
2. Inspector open on a scene with metadata and tags
3. Notes or Entities surface open with linked context visible
4. Sources / research browser with attachment and link context
5. Import / Export with compile preset and export review visible

## Theme Coverage

Capture at least one hero screenshot in each of:

- System or Light
- Parchment
- Midnight Ink
- Forest Draft
- Rose Study

## Capture Notes

- Prefer a maximized window on desktop resolution
- Keep sidebar and inspector visible unless the shot is editor-focused
- Use realistic sample manuscript content instead of empty placeholders
- Avoid clipping menus or sheets at the bottom edge
- Confirm color contrast for chips, highlights, and notices in each theme
- Use `Tools/focus_app_window.sh` to bring the app forward and size the window consistently before capture

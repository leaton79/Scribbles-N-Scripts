# Scribbles-N-Scripts v1.0.1

Date: March 11, 2026

## Summary

`v1.0.1` is the first public-facing polish release after the core `v1.0.0` milestone. It focuses on branding consistency, visual identity, themed presentation, and clearer first-run onboarding rather than new manuscript systems.

## Included

- Renamed and branded app shell carried fully through Scribbles-N-Scripts
- Bundled notebook-style app icon plus in-app brand mark usage
- Distinct workspace themes, quick theme switching, and deeper palette coverage across shell surfaces
- Theme-aware chips, highlights, metadata accents, and modular-mode styling
- Branded welcome/start screen for new and recent project entry
- Appearance presets for theme, font, editor width, and line spacing

## Verification

- `swift build`
- `swift test` with 346 passing tests

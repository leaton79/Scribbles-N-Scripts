# Scribbles-N-Scripts v1.0.0

Date: March 10, 2026

## Summary

`v1.0.0` marks the first milestone release of Scribbles-N-Scripts as a feature-complete manuscript-writing application across the core surfaces defined in the project Bible.

## Included

- Project creation, open, autosave, backups, restore, and recovery-mode open
- Rich writing workflow with editor, split editor, linear mode, modular mode, corkboard, and outliner
- Sidebar navigation, staging tray, command palette, inspector, and project settings
- Snapshots, tags, metadata schemas, goals/statistics, and advanced Find/Replace
- Timeline, entity tracking, notes, sources/research library, and scratchpad
- Import/export and compile support for Markdown, HTML, DOCX, PDF, and EPUB

## Verification

- `swift build`
- `swift test` with 344 passing tests

## Notes

- Automated screenshot-based visual QA may require macOS Accessibility and Screen Recording permissions for the host app or terminal.
- Remaining work is now primarily post-v1 depth, polish, and release hardening rather than missing core application shape.

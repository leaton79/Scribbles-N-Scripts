# Scribbles-N-Scripts Post-v1 Roadmap

Date: March 10, 2026

## Goal

Separate stable core product work from the next wave of feature depth and release hardening.

## Priority 1: Release Hardening

- Run a full keyboard-only and VoiceOver-oriented pass across all major surfaces
- Tighten consistency for empty states, notices, help text, and command discoverability
- Perform screenshot-driven visual QA once macOS permissions allow automated capture and resizing
- Do manual ship checks for recovery mode, export flows, staging, and project switching

## Priority 2: Research Workflow Depth

- Add in-app preview for common research attachment types
- Improve cross-navigation between sources, notes, entities, and scenes
- Strengthen citation review and source mention discovery
- Expand research-browser browsing and filtering for larger projects

## Priority 3: Publish Pipeline Depth

- Add richer rendered export previews for HTML, EPUB, and PDF
- Expand template and stylesheet management into a clearer publishing system
- Improve output review before export, including warnings and structure validation
- Continue polishing EPUB metadata, assets, and formatting consistency

## Priority 4: Future Product Upgrades

- Freeform corkboard positioning beyond the current grouped corkboard/outliner surfaces
- Richer inline writing assistance around entities, notes, and citations
- Additional release packaging and distribution work once the app is considered ship-ready

## Out Of Core v1 Scope

These are not blockers for the current stable core application:

- Deeper research-browser preview tooling
- More advanced publishing/template systems
- Additional visual polish beyond usability and consistency thresholds

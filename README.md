# Scribbles N Scripts

Scribbles N Scripts is a macOS writing application built with Swift and SwiftUI.

## Status

Early development. Core scaffolding and foundational modules are in progress.

## Requirements

- macOS 14+
- Xcode 15+ (or Swift 5.9+ toolchain)

## Build

```bash
swift build
```

## Test

```bash
swift test
```
